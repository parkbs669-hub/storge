[요청: 로컬 LLM 서버 최적화 가이드 제공]

내가 가지고 있는 PC 환경에서 로컬 LLM(LM Studio)을 가동하여 가장 빠른 추론 속도(토큰 생성 속도)를 내고 싶어. 시행착오 없이 처음부터 끝까지 한 번에 최적의 설정값을 뽑아내기 위해 다음 내 PC 사양에 맞는 세팅 가이드를 작성해 줘.

1. 내 시스템 사양 정보:
   - OS: Windows 10/11
   - GPU 모델: NVIDIA GeForce RTX 2070
   - 그래픽 메모리(VRAM): 8GB
   - 시스템 메모리(RAM): 16GB
   - 사용하고자 하는 메인 언어 모델 크기/종류: Qwen 3.5 9B 기반 모델

2. 아래 항목을 포함해서 구체적으로 알려줘:
   - VRAM 한계에 맞춘 최적의 모델 양자화 형식 추천 (GGUF 규격의 구체적 레벨)
   - LM Studio 하드웨어 옵션 세팅값 (GPU Offload 레이어 배분 수치, Context Length 값 제안)
   - 빠른 속도를 유도할 수 있는 최적의 추론 옵션 (Temperature, CPU 스레드 설정 등)
   - 속도를 극대화할 수 있는 추가적인 핵심 옵션 활성화 여부 (Flash Attention, KV 캐시 오프로드 설정 등)
